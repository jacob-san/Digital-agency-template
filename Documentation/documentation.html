Adding/editing an item on the home page

In order to change or add a position in the slider, you need to open one of the pages that you want to edit (index.html, index_v2.html, index_v3.html)

  • Project Category – Project category, for example Web-Site / Mobile App
  • Title – Just the name of the project.
  • Small description – A brief description in a few lines for a good balance.
  • path/your_image.jpg – Link to project image.
<!-- Caption 1 -->
<div class="swiper-slide">
    <div class="slider__item">
        <h6 class="title title--overhead"><span class="down-up"><span>Project Category</span></span></h6>
        <h1 class="title title--display-1 js-text-wave">Title</h1>
        <p class="description"><span class="down-up"><span>Small description</span></span></p>
        <a class="btn-link btn-link--circle-right" href="link.html"><span class="down-up"><span>See details<i class="circle circle--right icon-right-open"></i></span></span></a>
    </div>
</div>
<!-- /Caption 1 -->
...
<!-- Image 1 -->
<div class="swiper-slide">
    <div class="cover-slider" data-bg="path/your_image.jpg"><a class="swiper-slide__link" href="link.html"></a></div>
</div>
<!-- /Image 1 -->

If you want to change the delay between transitions, open the common.js file and find the line:

// settings
var $sliderDelay = 6000, // Delay between transitions (in ms).
$sliderSpeed = 1200; // Speed transitions

In the code, this line occurs four times, the first of them refers to the index.html page, the second to index_v2.html, the third to index_v2.html, the fourth to projects_single.html

Customization of contact form

In some cases, the form may not work. Contact your hosting support for problem solving.

You can also add or change form fields. To send mail using php library PHPMailer. Detailed documentation. For the form to work, a php version of 5.5 or higher is required.

To customize a contact form, you need to open the file form-contact.php from the folder php, find and replace line 78 on your emeil example@example.com.

$mail->addAddress('example@example.com', 'Company Name');

Here you can specify the subject of the message, line 83.

$mail->Subject = 'Contact Form';

DOM form element

  • id="" – Unique identifier for form element
  • name="" – The name of the field is intended so that the form handler can identify it
  • placeholder="" – Displays prompting text
  • required="required" – Required if the form field is required.
  • autocomplete="" – Enables or disables autocomplete: on/off 
<div class="form-group">
    <input type="email" class="input form-control" id="emailContact" name="emailContact" placeholder="Email address" required="required" autocomplete="on">
    <div class="help-block with-errors"></div>
</div>
Setting the marker on Google Map

Due to a change in Google policy, Google Maps has been replaced with alternative maps, but you can still add them. To do this, you will need to get the API key and follow the small instructions below.

1. Get API key

2. Open the contact-form.html page in the editor, find at the top of the line code (42 - 44) and delete them:

<!-- Mapbox-->
<script src='https://api.mapbox.com/mapbox-gl-js/v1.4.1/mapbox-gl.js'></script>
<link href='https://api.mapbox.com/mapbox-gl-js/v1.4.1/mapbox-gl.css' rel='stylesheet' />

3. Find the line (176 - 177) at the bottom of the page:

<!-- Mapbox init -->
<script src="assets/js/mapbox.init.js"></script>

Replaced by:

<!-- Google Maps -->
<script src="https://maps.googleapis.com/maps/api/js?key=YOUR-API-KEY"></script>
<script src="assets/js/google-map.init.js"></script>

You need to replace these coordinates with your own. To determine the coordinates you can visit the site Google Maps.

Setting on Mapbox

1. Register with Mapbox and get "Access tokens"

2. Open the mapbox.init.js file and paste your "Access Token" into the line (YOUR-ACCESS-TOKEN):

mapboxgl.accessToken = 'YOUR-ACCESS-TOKEN';

3. To configure your coordinates on the map, use the online mapbox maker

Enable RTL

1. To activate RTL in the template, open the assets/style/vendors/rtl folder. Copy the bootstrap.min file and paste it into the assets/style/vendors/ folder with a replacement.

2. Open all html pages in the editor (Sublime Text 3, Notepad ++ b etc.). After the line:

<link rel="stylesheet" type="text/css" href="style/style.css"/>

Paste:

<link rel="stylesheet" type="text/css" href="style/style-rtl.css"/>

3. Open the assets/js/rtl folder. Copy all three files and paste them into the js folder with a replacement.

Disable circle cursor

If you want to see a normal cursor, just open the file common.js, find line 231 and comment it out (see the example below "//")

//followCursor();
Disable inertial scroll

If you want to turn off smooth scrolling, then just open the common.js file, find line 98 and comment it out (see the example below "//")

//niceScroll();
